1. Create a database by manually or command line.
Create db DFTEST
Drop db DFTEST

2. Connect the Database by the following command line:
Connect to DFTEST user admin using 4ss3nt1sSERVER

3. Create the bufferpool and tablespace for the new database by the following command lines:
CREATE BUFFERPOOL LSO_BUFFERPOOL SIZE 8000 PAGESIZE 8192
CREATE TABLESPACE LSO_TABLESPACE PAGESIZE 8192 MANAGED BY DATABASE using (file 'D:\DB2\TABLESPACE\LSO_TABLESPACE0100001001 $N+100' 100000) BUFFERPOOL LSO_BUFFERPOOL
